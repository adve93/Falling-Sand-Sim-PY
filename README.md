# Falling Sand Sim PY
A basic falling sand simulation in python using pygames.
Use the mouse to drop selected material in the cursor position. Press S, R and E to enter Sand, Rock and Eraser Mode respectively.

Current features:
- Sand Particles.
- Rock Particles.
- Eraser Mode.
- A special cursos to display which Mode you are on.

Future Implementations:
- Containarize the App
- Water Particle
- Fire Particle
- Steam Particle
- Explosions
